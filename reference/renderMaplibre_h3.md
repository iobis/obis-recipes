# Shiny render function for `maplibre_h3`

Shiny render function for `maplibre_h3`

## Usage

``` r
renderMaplibre_h3(expr, env = parent.frame(), quoted = FALSE)
```

## Arguments

- expr:

  An expression that returns a `maplibre_h3` widget.

- env:

  The environment in which to evaluate `expr`.

- quoted:

  Logical. Is `expr` already quoted? Default: `FALSE`.

## Value

A Shiny render function.
