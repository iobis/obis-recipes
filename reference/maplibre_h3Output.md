# Shiny output function for `maplibre_h3`

Shiny output function for `maplibre_h3`

## Usage

``` r
maplibre_h3Output(outputId, width = "100%", height = "400px")
```

## Arguments

- outputId:

  Output variable to read from.

- width, height:

  Widget dimensions (CSS string or pixels).

## Value

A Shiny output object.
